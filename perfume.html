產品項目
香調

一種賦予新生命，如同走進充滿各式各樣花欉的大花園。

例如：玫瑰 / 茉莉 / 山茶 / 梔子 / 玉蘭 / 野薑 / 蓮花 / 康乃馨 / 桂花 / 鬱金香…等香精 。

給人帶來一種喜悅活潑舞動人生的新氣象。

例如：柑橘 / 檸檬 / 萊姆 / 白柚 / 蘋果(青/紅)…等香精。

一種給人清新氣味彷彿與大自然直接接觸的感覺。

例如：藤蔓 / 葉子 / 松葉 / 芬多精 / 草地…等香精。

青澀甘美純淨的香氣代領著一種無憂的心靈放鬆。

例如：烏龍茶香 / 紅茶香 / 綠茶香 / 青草茶香…等香精。

一種屬於清新自然的香氣給人純潔與白淨的感覺。

例如：小黃瓜 / 蜜瓜 / 佛手柑 / 蘆薈 / 白柚…等香精。

帶來一種沉穩及內斂之氣質。

例如：木香 / 檀香 / 檜木 / 麝香…等香精。

一種賦予廣大的冥想空間且獨一無二之專屬特定氣味。

例如：大自然的空氣 / 五行 / 海洋 / 大草原 / 森林浴…等香精。

另外透過生活上的特別需求，而衍生另一種別於一般香氣之變化。

例如：魚腥 / 腥臭 / 辣椒 / 辛香…等香精
